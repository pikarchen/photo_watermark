水印文件处理工具 - 项目结构说明
=====================================

📁 项目根目录
├── 📄 main.py              # 主程序文件，包含完整的应用逻辑
├── 📄 requirements.txt     # Python依赖包列表
├── 📄 README.md           # 项目说明文档
├── 📄 templates.json      # 水印模板配置文件
├── 📄 install.bat         # Windows安装脚本
├── 📄 run.bat            # Windows启动脚本
├── 📄 test_app.py        # 测试脚本
└── 📄 项目结构.txt        # 本文件

🔧 主要功能模块
===============

1. 主应用类 (WatermarkApp)
   - 界面布局和组件管理
   - 文件导入和列表管理
   - 水印设置和预览
   - 导出功能实现

2. 导出线程类 (ExportThread)
   - 后台批量处理图片
   - 进度更新和错误处理
   - 多线程安全操作

3. 核心功能
   - 图片格式支持：JPEG, PNG, BMP, TIFF
   - 文本水印：字体、颜色、透明度、样式
   - 图片水印：PNG透明通道支持
   - 位置控制：九宫格布局 + 点击切换
   - 模板管理：保存、加载、删除
   - 批量导出：进度显示、错误统计

🎨 界面布局
===========

左侧面板：
- 文件导入区域
- 图片列表显示
- 导出设置区域

中间面板：
- 图片预览区域
- 水印位置控制

右侧面板：
- 水印类型选择
- 文本水印设置
- 图片水印设置
- 模板管理

📋 使用流程
===========

1. 启动程序
   - 双击 run.bat 或运行 python main.py

2. 导入图片
   - 选择单张/批量/文件夹导入

3. 设置水印
   - 选择文本或图片水印
   - 调整参数和位置

4. 预览效果
   - 实时查看水印效果
   - 调整位置和参数

5. 导出图片
   - 选择输出格式和文件夹
   - 设置文件命名规则
   - 开始批量导出

🔧 技术栈
=========

- Python 3.7+
- PyQt5 (界面框架)
- Pillow (图像处理)
- JSON (配置存储)

📝 注意事项
===========

1. 首次运行需要安装依赖：pip install -r requirements.txt
2. 程序默认禁止导出到原文件夹
3. 模板文件保存在 templates.json 中
4. 支持透明通道的PNG图片作为水印
5. 所有设置变更都会实时预览
